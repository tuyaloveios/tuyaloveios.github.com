一个强大的网络框架

  • 简单易用

    BANetworking 提供了非常简单的API封装请求，能够让新手快速上手

  • 支持 Json 转 Object

    BANetworking 自带Json 转 Object类，并支持数据类型不匹配时自动过滤。

  • 接口分层

    只需按照 BANetworking 提供的分层思想可以帮助你管理好网络层的分层。

  • 丰富的接口

    BANetworking 提供了强大的基础接口，可以方便去操作HTTP头、Body体以及Cookie。
