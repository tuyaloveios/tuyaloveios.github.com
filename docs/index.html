什么是 BANetworking？

BANetworking 是一个简单、易用且高效的 iOS 网络框架。它基于NSURLSession上面做的一些封装，以帮助使用者模块化构建网络请求，使用非常简单。

BANetworking 主要分三个部分：网络、Model解析、数据存储。

安装

方法1：直接拖项目

  1. 去 github 上下载 BANetworking
  2. 解压后将BANetworking文件夹拖入项目工程中。

方法2：使用CocoaPods安装

如果没安装CocoaPods时请先安装CocoaPods，安装CocoaPods请参考 Mac下CocoaPods安装步骤
